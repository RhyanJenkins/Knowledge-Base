HERO Activites

When HERO is placed arrived on an event, on Info Page 2, the event will give you a drop down of different HERO activities to add to event. These will describe what the unit did while on scene.

When you select an activity, press the green "+" to add it to the vehicle. To remove an already added activity, select the red "X" .

The options in the drop down are as follows:

Description Of Activities:
Abandoned If vehicle is unoccupied
Air To fill low tires
Assist Law Enforcement Assisting LE with scene
Debris Clearing debris from roadway
Diesel Provide diesel fuel to vehicle
Dispatch Not Found Unable to find event
First Aid Provide basic first aid
Flat Tire Change a flat tire with motorists spare
Gas Provide unleaded fuel to vehicle
Jump Start Provide jump start to dead battery
Mechanical Any mechanical issue
MOT/Traffic Control ALWAYS a HERO activity, unless void or dispatch not found
Motorist Sleeping Motorist asleep in vehicle
No Assistance If motorist requests no assistance, or if vehicle pulls away before HERO can assist
Oil Provide oil to engine
Other Such as vehicle escort, help securing load in bed of truck, etc. Place note in history of event to describe your "Other."
Phone When motorist was using phone
Push Provide vehicle push to safe location
Slow Roll Provide slow roll to block/slow traffic so another unit/LE can move event to different location
Tools/Minor Repair Provide tools (jack, wrench, etc.) or minor repair to vehicle
Tow Request APD tow request for service/impound tow
Transport When HERO transports motorist(s) from scene to safe location. HERO must provide TMC with the following information. TMC must add information to event history:
  • Description of persons transported
  • Destination once transported
  • Starting and ending mileage of HERO truck
  • Advise when they are at specific location
Void When accidently placed wrong HERO unit on scene, use this to void them
Water Provide water for overheated engines
