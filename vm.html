Vehicle Manger

When HERO calls in service for the day, you will need to add them to the Vehicle Manager in Lonestar. When HERO goes Out of Service for the day, you will need to remove them from the Vehicle Manager.

To add/remove HERO, first go to the vehicles row. Under each column select the arrow and then select the item you want. This will make a timestamp for that vehicle. It is important to be timely accurate, in case reports need to be pulled to see timestamps for a particular vehicle.

*Telephone numbers for vehicles can be found in the HERO Operational Information sent by Jeff Morton*


To filter so you only see the units working a particular shift, select the pin on the shift column and select the shift you would like to filter. It will then only show you the vehicles on the shift(s) selected.


When a HERO unit calls in an event, you open the event through the Vehicle Manager. There are two ways to open an event from the Vehicle Manager:

Select the vehicle you want to place on the event. Then click "Create An Event". This will open the event and place that unit on scene.

Select the vehicle you want to place on the event. Right click, and select "Create Event".
