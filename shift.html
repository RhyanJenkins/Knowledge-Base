HERO Shifts Start and End

Start of HERO Shift:

See Vehicle Manager tab on how to add HERO units when they come In Service.

First Check:

MON-FRI – first and second shift, first hour of shift (First shift: 5-6AM, Second shift: 1-2pm)

SAT/SUN – HERO units for the day (between 7-8AM)

0-60 minutes after start of HERO shift, all events on route must be checked on.
*Note: all events should be checked on throughout the day, every 2 hours.
This page explains the first check to be done in the morning and afternoon.

Abandoned Vehicles

-If on CCTV, verify that vehicle is still there and add note in history of event. Keep watching and update event accurately until vehicle has cleared. You can advise HERO that you have this abandoned vehicle on camera and they do not need to stop if HERO has already marked vehicle.

-If not on CCTV, verify with HERO if vehicle is still on scene and still abandoned.
*Be advised, HERO only patrols IH-35 on weekends. If you work weekends and have an abandoned vehicle on another route, make a note in the event history that HERO is unable to check on vehicle on weekends due to HERO only patrolling IH-35. Monday morning all vehicles should be checked on all routes.

If vehicle has been verified on scene for 48 hours, the appropriate law enforcement agency must be notified of abandoned vehicle. When calling obtain:

  • Case number, if available.
  • Operator spoken to.

Add a note in the event history of this information.

If vehicle is still on scene after another 48 hours, call LE agency back and ask for update. Again, obtain the operator of who you spoke to and add another note in event history. Do so until vehicle has cleared.

Events found on CCTV or called in when HERO was OOS:

  • Make a note in event history that HERO was unable to be dispatched earlier due to HERO OOS
  • If found on CCTV, verify that event is ongoing
  • Dispatch HERO to event as soon as they come in service. Advise them of as much information as you can

End of HERO Shift:

See Vehicle Manager tab on how to remove HERO units when they go OOS for the day.

FINAL CHECK:

MON-FRI – second shift HERO units - between 8PM-9PM.
SAT/SUN – HERO units for the day - between 6PM-7PM.

0-60 minutes before HERO goes OOS for the day, all remaining events on route must be checked on.
*Note: all events should be checked on throughout the day, every 2 hours.
This page explains the final check to be done in the afternoon.

Abandoned Vehicles:


-If on CCTV, verify that vehicle is still there and add note in history of event. Keep watching and update event accurately until vehicle has cleared.

-If not on CCTV, verify with HERO if still on scene and still abandoned.

If vehicle has been verified on scene for 48 hours, the appropriate law enforcement agency must be notified of abandoned vehicle. When calling obtain:

  • Case number, if available
  • Operator spoken to

Add a note in the event history of this information.

If vehicle is still on scene after another 48 hours, call LE agency back and ask for update. Again, obtain the operator of who you spoke to and add another note in event history. Do so until vehicle has cleared.

Disabled Vehicles:

Verify with HERO if vehicle is still occupied.
-If still occupied, law enforcement in area must be notified.

-If on CCTV, keep the event open and accurate until cleared.

-If not on CCTV, after notifying LE and adding proper notes, close out event.

-The event is now LE’s responsibility. If vehicle is still there in the morning per HERO, create a new event.

-If no longer occupied, continue to treat as an abandoned vehicle. (See above)
